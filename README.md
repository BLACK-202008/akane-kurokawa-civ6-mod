# 《我推的孩子》黑川茜 - Civilization VI 自定义文明模组

这是一个为《Sid Meier's Civilization VI》制作的自定义文明模组。模组新增可游玩文明 `LALALAI` 与领袖 `黑川茜`，整体设计围绕“文化主导，但能向多体系外溢”的思路展开：
前期依靠独特剧团稳住文化底盘，中期通过话剧演员和城市联动滚起节奏，后期再根据局势在信仰、旅游、科技、生产和军事之间切换重心。

项目当前已经完成核心玩法实现、基础 UI 表现和主要回归修复，能够正常进入游戏并进行功能测试。

## 模组定位

- 主题来源：`《我推的孩子》` 中的 `黑川茜`
- 文明名称：`LALALAI`
- 领袖名称：`黑川茜`
- 玩法定位：`文化核心`、`多体系联动`、`中后期强势成长`
- 适用版本：`Rise and Fall` / `Gathering Storm`

这个文明不是单纯的“文化胜利特化模板”。它更接近一个以文化为发动机的复合型文明：

- 能用文化和宜居度快速建立发展节奏
- 能把剧团周围的专业区域一起带强
- 能通过领袖模式切换应对不同局势
- 既有旅游与伟人玩法，也保留可观的生产和作战能力

## 核心特色

### 1. 特色区域：LALALAI 剧团

`LALALAI 剧团` 是替代剧院广场的特色区域，是整个文明的中轴。

当前实现重点：

- 半价区域，便于前中期快速铺开
- 自带 `+2 文化` 与 `+2 宜居度`
- 相邻 `市中心` 与 `奇观` 时，额外获得 `+2 金币`
- 对相邻专业区域提供外溢相邻加成：
  - 学院：`+1 科技`
  - 商业中心：`+1 金币`
  - 港口：`+1 金币`
  - 军营：`+1 生产力`
  - 工业区：`+1 生产力`
  - 圣地：`+1 信仰`

这使得 `LALALAI 剧团` 不只是文化区域，而是能主动参与城市规划的“枢纽区域”。你通常会希望围绕它布局学院、商业中心或军营，让单城形成更高密度的联动。

### 2. 文明能力：多面舞台术

文明能力的核心逻辑是“剧团不仅给相邻加成，还把相邻区域内的建筑和伟人体系一起带起来”。

当前实现重点：

- 所有从 `LALALAI 剧团` 获得相邻加成的区域，其相关建筑产出 `+20%`
- 与 `LALALAI 剧团` 相邻的以下区域会额外获得对应伟人点数：
  - 学院：大科学家点数
  - 商业中心：大商人点数
  - 港口：大海军统帅点数
  - 军营：大将军点数
  - 圣地：大预言家点数

这套设计让城市布局价值被放大。一个围绕剧团构建的高联动城市，不仅基础产出更高，伟人竞争力也会更强。

### 3. 特色单位：话剧演员

`话剧演员` 是黑川茜文明专属的平民单位，当前已经不再作为原版 `摇滚乐队` 的替代品存在，也不再拥有摇滚乐队的音乐会能力或升级链。它的玩法完全围绕模组自定义的“舞台演出”机制。

当前实现重点：

- 通过信仰购买
- 拥有 `3` 次演出机会
- 只能在己方城市的 `LALALAI 剧团` 上执行演出
- 每座城市只能被演出一次
- 演出后：
  - 该城市永久获得 `+1 文化`
  - 若该城已有剧院系建筑，则对应建筑获得额外文化与旅游强化
  - 同时触发全国范围、可叠加的 `艺术繁荣`

这个单位的用途不是去外面刷场地，而是像“文明内部的成长引爆器”。它更偏经营型资源，而不是原版摇滚乐队那种一次性外部投放单位。

### 4. 全国状态：艺术繁荣

每次 `话剧演员` 演出，都会给整个文明增加一层 `艺术繁荣`。

当前实现重点：

- 作用范围：`全国`
- 效果：所有城市区域产出 `+5%`
- 持续时间：`20 回合`
- 支持叠加：再次演出会继续增加层数并延长持续时间
- 已处理读档恢复与到期清算

这让话剧演员的每次演出都不只是“单城成长”，而是全国节奏强化。越早开始运作，滚雪球越明显。

## 领袖：黑川茜

黑川茜的核心是“角色切换”。她可以在不同模式之间切换，把文明当前的重心转到最需要的方向。

### 基础能力：推理大师

无论处于哪种模式，黑川茜始终拥有以下基础能力：

- 固定 `+1 文化`
- 与 `LALALAI 剧团` 相邻的学院额外获得 `+1 科技`
- 解锁 `数学` 后，额外获得 `+10% 科技`

这部分保证了领袖即使还没完成模式切换，也始终有稳定收益，而且能自然强化“剧团带学院”的城市布局。

### 可切换模式

#### 星野爱模式

偏向信仰与宗教起势。

- `+1 信仰`
- 大预言家点数固定 `+1`
- 每个圣地区域额外 `+1` 大预言家点数
- `+30% 信仰`

适合需要抢宗教、补信仰经济或利用信仰购买时切入。

#### 鞘姬模式

偏向文化与旅游推进。

- `+1 文化`
- 大艺术家点数固定 `+1`
- 每个 `LALALAI 剧团` 额外 `+1` 大艺术家点数
- `+20% 文化`
- `+30% 旅游业绩`

这是最适合文化胜利推进和中后期爆发的一档模式，和话剧演员、艺术繁荣、剧院建筑强化配合非常紧。

#### 星野露比模式

偏向作战与生产爆发。

- 所有军事单位 `+5 战斗力`
- 军事单位经验获取 `+20%`
- 大将军点数固定 `+1`
- 每个军营额外 `+1` 大将军点数
- `+20% 生产力`

这个模式保证了文明不是文化白板。需要打仗、反打或军备转型时，可以直接切过去。

### 模式切换机制

当前实现重点：

- 模式切换有冷却
- 标准速度下冷却为 `10 回合`
- 其他速度按规则换算
- 切换成功后立刻获得：
  - `+8% 全产出`
  - 持续冷却时长的一半
  - 一笔按当前回合数缩放的金币奖励

因此切模式不是单纯“改配置”，而是一个真正能改变当前节奏的爆发按钮。

## 适合的玩法方向

这个文明比较适合以下几种思路：

### 文化滚雪球

- 优先铺 `LALALAI 剧团`
- 利用剧院建筑与话剧演员快速把文化做高
- 中后期切入 `鞘姬模式`

### 发育转综合

- 前期用剧团带学院、商业中心、圣地
- 中期靠文明能力把多区域建筑产出放大
- 视局势在信仰、科技、经济间灵活切换

### 文化底盘转军事

- 先用文化提速城市建设与制度推进
- 中后期切入 `星野露比模式`
- 借高生产和 `+5 战斗力` 打出压制

## 技术结构

项目主要分为以下部分：

- `Data/`
  - 文明、领袖、区域、单位、修饰符、需求集等数据库定义
- `Scripts/`
  - 话剧演员演出逻辑
  - 艺术繁荣状态维护
  - 黑川茜模式切换系统
- `UI/`
  - 顶栏模式切换界面
  - 单位面板演出按钮注入
  - 生产与城市界面兼容处理
- `Text/`
  - 中英文文本
- `Icons/` / `Textures/` / `ArtDefs/`
  - 图标、美术资源与表现映射
