# 《我推的孩子》黑川茜 - Civilization VI 自定义文明模组

这是一个为《Sid Meier's Civilization VI》制作的自定义文明模组。模组新增可游玩的文明 `LALALAI` 与领袖 `黑川茜`，整体设计围绕“文化为核心，但能向信仰、科技、生产与作战外溢”的思路展开。

项目前期依靠 `LALALAI 剧团` 建立文化与城市联动，中期通过 `话剧演员` 和全国性增益滚动成长，后期再通过黑川茜的三种模式切换，在信仰、旅游与战争之间调整节奏。

## 模组定位

- 主题来源：《我推的孩子》中的 `黑川茜`
- 文明名称：`LALALAI`
- 领袖名称：`黑川茜`
- 玩法定位：`文化核心`、`多区域联动`、`中后期切换爆发`
- 适用版本：`Rise and Fall` / `Gathering Storm`

这不是一个只会堆文化的单线文明。它更接近一个以文化区为中轴、能把相邻区域、伟人点、信仰经济和战斗节奏一起拉起来的复合型文明。

## 核心特色

### 1. 特色区域：LALALAI 剧团

`LALALAI 剧团` 是替代剧院广场的特色区域，也是整个文明的布局核心。

当前实现重点：

- 自带 `+2 文化` 与 `+2 宜居度`
- 相邻 `市中心` 或 `奇观` 时，额外获得 `+2 金币`
- 对相邻专业区域提供额外相邻加成：
  - 学院：`+1 科技`
  - 商业中心：`+1 金币`
  - 港口：`+1 金币`
  - 军营：`+1 生产力`
  - 工业区：`+1 生产力`
  - 圣地：`+1 信仰`
- 该区域每个专家额外提供 `+2 食物` 与 `+1 生产力`

这使得 `LALALAI 剧团` 不只是文化区，而是能主动参与城市布局的“枢纽区”。围绕它摆学院、商业中心、军营或圣地，能把单城价值放大很多。

### 2. 文明能力：多面舞台术

文明能力的核心是“剧团不只给相邻加成，还把相邻区域的建筑产出和伟人体系一起带起来”。

当前实现重点：

- 所有从 `LALALAI 剧团` 获得相邻加成的区域，其相关建筑产出 `+50%`
- 与 `LALALAI 剧团` 相邻的以下区域会额外获得对应伟人点数，并会随着这些区域的建筑等级继续提高：
  - 学院：大科学家点数
  - 商业中心：大商人点数
  - 港口：大海军统帅点数
  - 军营：大将军点数
  - 圣地：大预言家点数

这套设计会显著放大城市规划价值。一个围绕剧团搭建的高联动城市，不仅基础产出更高，伟人竞争力也会更强。

这里的“相关建筑产出提高”指的是对应区域建筑本身的产出会被放大，不是整座城市同类总产出一起提高。当前实现按 `LALALAI` 实际会建造的标准建筑链逐项覆盖，不额外兼容其他文明的特色替代建筑。

### 3. 特色单位：话剧演员

`话剧演员` 是黑川茜文明的专属平民单位。它已经不再作为原版 `摇滚乐队` 的替代品存在，也不再继承摇滚乐队的音乐会、升级链或随机推广玩法；它完全围绕模组自定义的“舞台演出”机制工作。

当前实现重点：

- 通过信仰购买
- 拥有 `3` 次演出机会
- 只能在己方城市的 `LALALAI 剧团` 上执行演出
- 每座城市只能被演出一次
- 演出后：
  - 该城市永久获得 `+1 文化`
  - 若该城已有剧院系建筑，则对应建筑获得额外文化与旅游强化
  - 同时触发全国范围、可叠加的 `艺术繁荣`

它的定位不是外出刷单次收益，而是作为文明内部的成长引擎，把单城成长和全国节奏一起往上推。

### 4. 全国状态：艺术繁荣

每次 `话剧演员` 演出，都会给整个文明增加一层 `艺术繁荣`。

当前实现重点：

- 作用范围：`全国`
- 效果：所有城市产出 `+8%`
- 持续时间：标准速度 `25 回合`，并会根据游戏速度折算
- 持续上限：标准速度 `100 回合`，并会根据游戏速度折算
- 支持叠加：再次演出会继续增加层数并延长持续时间
- 已处理读档恢复与到期清算

这让话剧演员的每次演出都不只是“单城成长”，而是全国节奏强化。越早开始滚动，收益越明显。

## 领袖：黑川茜

黑川茜的核心是“角色切换”。她可以在不同模式之间切换，把文明当前的重心转到最需要的方向。

### 基础能力：推理大师

无论处于哪种模式，黑川茜始终拥有以下基础能力：

- 固定 `+1 文化`
- 与 `LALALAI 剧团` 相邻的学院额外获得 `+1 科技`
- 解锁 `数学` 后，额外获得 `+10% 科技`
- 解锁 `科学理论` 后，额外再获得 `+15% 科技`

这部分保证了领袖即使还没切模式，也始终有稳定收益，而且会自然强化“剧团带学院”的城市布局。

### 可切换模式

#### 星野爱模式

偏向信仰与宗教起势。

- `+1 信仰`
- 大预言家点数固定 `+1`
- 每个圣地区域额外 `+1` 大预言家点数
- `+30% 信仰`
- 所有宗教单位与审判官在宗教战斗中 `+10 战斗力`

适合需要抢宗教、补信仰经济，或要在宗教对抗里压住对手时切入。

#### 鞘姬模式

偏向文化与旅游推进。

- `+1 文化`
- 大艺术家点数固定 `+1`
- 每个 `LALALAI 剧团` 额外 `+1` 大艺术家点数
- `+20% 文化`
- `+30% 旅游业绩`
- 每次伟人隐退时，额外获得 `1` 个随机尤里卡

这是最适合文化胜利推进和中后期爆发的一档模式，和话剧演员、艺术繁荣、剧院建筑强化配合很紧。

#### 星野露比模式

偏向作战与生产爆发。

- 所有单位 `+5 战斗力`
- 所有单位 `+1 移动力`
- 单位经验获取 `+20%`
- 大将军点数固定 `+1`
- 每个军营额外 `+1` 大将军点数
- `+20% 生产力`

这个模式保证了文明不是单纯的文化白板。需要打仗、反打或军备转型时，可以直接切过去。

### 模式切换机制

当前实现重点：

- 模式切换有冷却
- 标准速度下冷却为 `10 回合`
- 其他速度按规则换算
- 切换成功后立刻获得：
  - `+15% 全产出`
  - 持续当前冷却时长的一半
  - 一笔按当前回合数缩放的金币奖励

因此切模式不是单纯“改配置”，而是一个真正能改变当前节奏的爆发按钮。

## 适合的玩法方向

### 文化滚雪球

- 优先铺 `LALALAI 剧团`
- 利用剧院建筑与 `话剧演员` 快速把文化做高
- 中后期切入 `鞘姬模式`

### 发育转综合

- 前期用剧团带学院、商业中心、圣地
- 中期靠文明能力把多区域建筑产出一起放大
- 视局势在信仰、科技、经济间灵活切换

### 文化底盘转军备

- 先用文化提速城市建设与制度推进
- 中后期切入 `星野露比模式`
- 借高生产、移动力与战斗力完成压制

## 技术结构

项目主要分为以下部分：

- `Data/`
  - 文明、领袖、区域、单位、Modifier、Requirement 等数据库定义
- `Scripts/`
  - 话剧演员演出逻辑
  - 艺术繁荣状态维护
  - 黑川茜模式切换系统
  - 鞘姬模式随机尤里卡逻辑
- `UI/`
  - 顶栏模式切换界面
  - 单位面板演出按钮注入
  - 生产与城市界面兼容处理
- `Text/`
  - 中英文文本
- `Icons/` / `Textures/` / `ArtDefs/`
  - 图标、美术资源与表现映射

## 安装与测试

1. 将源码目录内容同步到游戏测试目录
2. 在 Civilization VI 的模组菜单中启用本模组
3. 新开一局，选择 `LALALAI / 黑川茜`
4. 重点验证：
   - `LALALAI 剧团` 的相邻加成、专家产出与金币加成
   - `话剧演员` 的自定义演出
   - `艺术繁荣` 的全国叠层与到期
   - 三种领袖模式的切换、冷却与奖励
   - 星野爱模式的宗教单位战斗加成
   - 鞘姬模式的大人物隐退随机尤里卡
   - 读档后的状态恢复
