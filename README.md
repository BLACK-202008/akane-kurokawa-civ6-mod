# 《我推的孩子》黑川茜 - Civilization VI 自定义文明模组

这是一个为《Sid Meier's Civilization VI》制作的自定义文明模组。模组新增可游玩的文明 `LALALAI` 与领袖 `黑川茜`，玩法核心不是单纯堆文化，而是围绕 `LALALAI 剧团` 这个枢纽区，把文化区布局延伸到科技、信仰、生产、伟人点与作战节奏。


## 模组定位

- 主题来源：《我推的孩子》中的 `黑川茜`
- 文明名称：`LALALAI`
- 领袖名称：`黑川茜`
- 玩法关键词：`区域联动`、`剧院强化`、`阶段爆发`、`模式切换`
- 适用版本：`Rise and Fall` / `Gathering Storm`

这套设计的目标不是让文明“什么都能做一点”，而是让它通过同一套城市布局，在不同时期切换重心：

- 前中期靠 `LALALAI 剧团 + 学院/商业/圣地/军营` 做多区联动
- 中后期靠 `话剧演员` 把重要城市做成高联动核心城，同时开始爆发。
- 根据局势切入 `星野爱`、`鞘姬`、`星野露比` 三种模式

## 文明核心

### 1. 特色区域：LALALAI 剧团

`LALALAI 剧团` 替代原版 `剧院广场`，是整个文明玩法的中心。

当前实现的具体效果如下：

- 基础提供 `+1 [ICON_Culture] 文化`
- 基础提供 `+2 宜居度`
- 保留 `xp2` 规则下 `剧院广场` 原有的 `奇观 / 相邻区域 / 政府广场 / 娱乐区域` 文化相邻加成
- 每相邻 `1` 个 `市中心`，获得 `+1 [ICON_Culture] 文化`，并额外获得 `+2 [ICON_Gold] 金币`
- 每相邻 `1` 个 `奇观`，获得 `+2 [ICON_Culture] 文化`，并额外获得 `+2 [ICON_Gold] 金币`
- 每相邻 `1` 个 `娱乐中心`、`水上乐园` 或其对应替代区域，获得 `+2 [ICON_Culture] 文化`
- 相邻的以下专业区域会从 `LALALAI 剧团` 获得额外相邻加成：
  - 学院：`+1 [ICON_Science] 科技`
  - 商业中心：`+2 [ICON_Gold] 金币`
  - 港口：`+2 [ICON_Gold] 金币`
  - 娱乐中心：`+2 [ICON_Production] 生产力`
  - 水上乐园：`+2 [ICON_Production] 生产力`
  - 军营：`+1 [ICON_Production] 生产力`
  - 工业区：`+1 [ICON_Production] 生产力`
  - 圣地：`+1 [ICON_Faith] 信仰`
- 该区域专家额外提供：
  - `+2 [ICON_Food] 食物`
  - `+1 [ICON_Production] 生产力`

这意味着 `LALALAI 剧团` 不是单纯堆文化的区域，而是城市布局的核心枢纽。它一方面保留了 xp2 规则下剧院广场原本的文化相邻骨架，另一方面又把市中心、奇观和周边专业区域一起卷进收益链，让文化、金币与跨区联动同时放大。

### 2. 文明能力：多面舞台术

文明能力的重点不是给剧团一条新数值，而是让它把相邻区域的建筑链和伟人链一起带起来。

当前实现的具体效果如下：

- 与 `LALALAI 剧团` 相邻的以下区域，其标准建筑链产出提高 `+50%`
  - 学院建筑链：图书馆、大学、研究实验室
  - 商业中心建筑链：市场、银行、证券交易所
  - 港口建筑链：灯塔、造船厂、海港
  - 军营建筑链：兵营、马厩、军械库、军事学院
  - 工业区建筑链：工坊、工厂、电子工厂、发电厂
  - 圣地建筑链：神社、寺庙
- 与 `LALALAI 剧团` 相邻的以下区域，会额外获得对应伟人点数
  - 学院：大科学家点数
  - 商业中心：大商人点数
  - 港口：大海军统帅点数
  - 军营：大将军点数
  - 圣地：大预言家点数
- 上述伟人点不是只给区域本体，而是会随着该区域建筑等级继续增加
  - 例如学院不仅本体加点，图书馆、大学、研究实验室也会继续补额外点数
  - 军营同理，兵营/马厩、军械库、军事学院都会继续抬高将军点

因此这个文明强的不是“单个区域特别夸张”，而是你只要把城布好，一座核心城会在建筑、伟人、相邻加成三个维度一起起飞。

### 3. 特色单位：话剧演员

`话剧演员` 是黑川茜文明的专属平民单位。它已经不再继承原版 `摇滚乐队` 的音乐会、升级链或随机推广逻辑，而是完全服务于模组自己的“舞台演出”机制。

当前实现的具体效果如下：

- 可使用 `信仰` 购买
- 拥有 `3` 次演出机会
- 只能在己方城市的 `LALALAI 剧团` 上执行演出
- 每座城市最多只能被演出 `1` 次
- 每次演出都会消耗 `1` 次演出机会
- 单位仅在最后一次演出后移除

演出后的城市强化效果为：

- 该城市的 `LALALAI 剧团` 及与其相邻的专业区域，相邻加成都会翻倍
  - 这同时包含 `xp2` 规则下的原版相邻加成与 `LALALAI` 提供的额外相邻加成
  - 对 `LALALAI 剧团` 来说，会一起翻倍的包括：
    - 来自 `市中心` 的 `+1 [ICON_Culture] 文化`
    - 来自 `奇观` 的 `+2 [ICON_Culture] 文化`
    - 来自 `娱乐中心`、`水上乐园` 或其替代区域的 `+2 [ICON_Culture] 文化`
    - 来自 `政府广场` 的 `+1 [ICON_Culture] 文化`
    - 来自 `市中心 / 奇观` 的额外 `+2 [ICON_Gold] 金币` 等
  - 对相邻的学院、商业中心、港口、娱乐中心、水上乐园、军营、工业区、圣地来说，其当前已有的相邻产出也会整体翻倍
- 如果该城已有剧院系建筑，则对应建筑继续获得额外强化
  - `古罗马剧场`：额外 `+1 [ICON_Culture] 文化`
  - `美术馆`：额外 `+1 [ICON_Culture] 文化`
  - `考古博物馆`：额外 `+1 [ICON_Culture] 文化`
  - `广播中心`：额外 `+1 [ICON_Culture] 文化`
- 该城市每拥有 `1` 座上述建筑，`旅游业绩` 额外提高 `+20%`
- 同时触发全国范围、可叠加的 `艺术繁荣`

还有一个实现细节很重要：

- 演出不是只在按下按钮那一刻结算一次
- 如果这座城市之后新修出了与剧团相邻的专业区域，或者在剧团旁新落成奇观，脚本会继续补挂缺失的对应收益

这让 `话剧演员` 更像是在“指定一座核心城进入高联动状态”，而不是拿一次性奖励就结束。

### 4. 全国状态：艺术繁荣

每次 `话剧演员` 演出，都会给整个文明增加一层 `艺术繁荣`。

当前实现的具体效果如下：

- 作用范围：`全国`
- 每层提供：
  - `+8% [ICON_Food] 食物`
  - `+8% [ICON_Production] 生产力`
  - `+8% [ICON_Gold] 金币`
  - `+8% [ICON_Science] 科技`
  - `+8% [ICON_Culture] 文化`
  - `+8% [ICON_Faith] 信仰`
- 标准速度持续时间：`25 回合`
- 标准速度持续上限：`100 回合`
- 其他游戏速度会按脚本比例折算
- 支持叠层
- 支持读档恢复
- 到期后会自动清算

这套状态保证了 `话剧演员` 不是只服务单城。你前期越早开始叠层，后面越容易把全国节奏一起顶上去。

## 领袖：黑川茜

黑川茜的核心设计是“有稳定科技底盘，但真正的节奏由模式切换决定”。基础能力不再是平铺文化，而是更明确地服务于 `学院联动` 与 `中后期爆发`。

### 基础能力：推理大师

无论处于哪种模式，黑川茜始终拥有以下基础能力：

- 固定获得 `+1 [ICON_Culture] 文化`
- 解锁 `数学` 后，额外获得 `+10% [ICON_Science] 科技`
- 解锁 `科学理论` 后，额外再获得 `+10% [ICON_Science] 科技`
- 每次伟人隐退时，额外获得 `1` 个随机尤里卡

## 三种模式

### 1. 星野爱模式

定位：偏向 `信仰经济`、`宗教对抗`

当前实现的具体效果如下：

- `+1 [ICON_Faith] 信仰`
- 大预言家点数固定 `+1`
- 每个圣地区域额外 `+1` 大预言家点数
- `+30% [ICON_Faith] 信仰`
- 所有宗教单位与审判官在宗教战斗中 `+10 战斗力`
- 使用信仰购买单位或建筑时，返还相当于本次信仰消耗 `80%` 的金币

### 2. 鞘姬模式

定位：偏向 `文化推进` 、 `旅游兑现`、`文化鼓舞`

当前实现的具体效果如下：

- `+1 [ICON_Culture] 文化`
- 大艺术家、大作家、大音乐家点数固定均为 `+1`
- 每个 `LALALAI 剧团` 额外分别 `+1` 大艺术家、大作家、大音乐家点数
- `+20% [ICON_Culture] 文化`
- `+30% 旅游业绩`
- 每次招募伟人时，额外获得 `1` 个当前可推进市政中的随机鼓舞

### 3. 星野露比模式

定位：偏向 `作战强压` 与 `军备爆发`

当前实现的具体效果如下：

- 所有单位 `+5 [ICON_Strength] 战斗力`
- 所有单位 `+1 移动力`
- 固定 `+1 [ICON_Production] 生产力`
- 大将军点数固定 `+1`
- 每个军营区域额外 `+1` 大将军点数
- 每个军营区域额外 `+1 [ICON_Production] 生产力`
- 击败敌方单位时，额外获得相当于该单位 `80% Cost` 的 `[ICON_Faith] 信仰`


## 模式切换机制

模式切换仍然是黑川茜的关键爆发按钮。

当前实现的具体规则如下：

- 标准速度冷却：`10 回合`
- 快速/联网/史诗/马拉松会按脚本规则折算
- 切换后会立即进入冷却
- 切换成功后，在“当前冷却时长的一半”期间获得：
  - `+16% [ICON_Food] 食物`
  - `+16% [ICON_Production] 生产力`
  - `+16% [ICON_Gold] 金币`
  - `+16% [ICON_Science] 科技`
  - `+16% [ICON_Culture] 文化`
  - `+16% [ICON_Faith] 信仰`


## 技术结构

项目主要由以下部分组成：

- `Data/`
  - 文明、领袖、区域、单位、Modifier、Requirement 等数据库定义
- `Scripts/`
  - `话剧演员` 演出逻辑
  - `艺术繁荣` 叠层、持续时间、到期清算
  - 模式切换冷却与切换爆发
  - 星野爱模式的信仰购买返金
  - 推理大师的伟人隐退随机尤里卡
  - 鞘姬模式的伟人招募随机鼓舞
  - 演出后城市额外相邻收益的持续维护
- `UI/`
  - 顶栏模式切换界面
  - 单位面板演出按钮注入
  - 相关 UI 兼容处理
- `Text/`
  - 中英文文本
- `Icons/` / `Textures/` / `ArtDefs/`
  - 图标、美术资源与表现映射
